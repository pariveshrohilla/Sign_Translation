If you wanna use this particular code then follow the particular steps

1. Install python version in between 8 and 11 for tensorflow to work
2. pip install tensorflow
3. pip install mediapipe
4. Create a folder names as Images with subfolders of all alphabets and digits
5. copy collectdata.py and run it to save your images into a file named as Images 
6. Capture a few images and save them
7. copy and run function.py to run all the necessary functions
8. copy and run data.py to save the end points of your hand and save thema as npy format for each alphabet
9. Copy and run trainmodel.py to run the model and train your model
10. If you want you can change the epoch size in a range of 200 and 500
11. A new file will be automatically saved model.h5 where your model will be saved
12. If your accuracy is greater than 0.8 then go ahead
13. Copy and run app.py to run the final code as required
